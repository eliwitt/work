We can create what is called a “Self-hosting” application that starts up the WCF service. Self-hosting is the simplest way to host services. You write the code necessary to initialize the ServiceHost and manage the service’s lifetime. Console, WinForm, WPF, and windows services are examples of the types of projects that I am talking about. For this project you have to add references to all three libraries from above. The System.ServiceModel, interface DemoInterfaces, and DemoCustomerBz are the libraries.

My first section of the demo for scroll items

I will place different items here that I feel does not fall into one of the other topics.

I created a library that holds different classes one of those classes’ uses my customized collection class. In the “CompareValues” method which you have to implement I compare values from different fields so I have to know the field’s type.

If TypeOf valueOfX Is Double Then Dim myDbl As Double Double.TryParse(ValueToFind.ToString, myDbl) Dim comp As IComparable = CType(valueOfX, IComparable) valCompare = comp.CompareTo(myDbl)

So “valCompare” holds 0=Equal, 1=Greater, or -1=Less which you can then decide with the additional help of a constraint variable to determine if you have a match. Note, you could use this same check in validation. Hook up an event. AddHandler Me._operation.ProgressChanged, AddressOf _oper_ProgressChanged A subreport event handler for an RDLC report.

My second section of the demo for scroll items

You can use LINQ with tableadapters. There are two ways you can do this the first is to use the Field collection that the fill method populates as follows: this.sAP2ChampsRunTableAdapter.Fill(this.sparePartsDS.SAP2ChampsRun); var partDt = (from partsrc in this.sparePartsDS.SAP2ChampsRun.AsEnumerable() select partsrc.Field("PartsUpDt")).FirstOrDefault(); this.tslPartDate.Text = "Parts as of " + partDt.Date.ToShortDateString();

Then the other way is to use the datatable and datarow collection that the GetData(type safe) method populates.

DataSets.SparePartsDS.SAP2ChampsRunDataTable myParts = this.SAP2ChampsRunTableAdapter.GetData(); var partII = (from partSrc in myParts.AsEnumerable() select partSrc.PartsUpDt).FirstOrDefault(); this.lblPartsUpDt.Text = "Parts as of " + partII.Date.ToShortDateString();
